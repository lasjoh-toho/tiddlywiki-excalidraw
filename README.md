# tw-excalidraw

A whiteboard plugin using Excalidraw.

Demonstration and documentation at <https://lasjoh-toho.github.io/tiddlywiki-excalidraw>.

## Development

### Installing dependencies

Please only use pnpm, as this project depends on pnpm-specific functionality.

```
$ pnpm install
```

### Development server

#### Read-only wiki

```
$ pnpm run dev
```

#### Editable wiki

```
$ pnpm run dev:wiki
```

### Editing

#### Formatting

```
$ pnpm run format
```

#### Linting

```
$ pnpm run lint
```

#### Type checking

```
$ pnpm run check
```

### Publishing

#### Library

```
$ pnpm run build
```

#### Online wiki

```
$ pnpm run publish
```
